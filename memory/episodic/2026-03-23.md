- ✅ 必须用 `read` 验证内容是否成功写入

**修复流程**：
1. ✅ 使用 `read` 确认文档为空
2. ✅ 使用 `write` 写入完整内容
3. ✅ 使用 `read` 验证内容成功写入（block_count: 108）
