### 🔥 重要教训：威权政权韧性（2026-03-21）

**案例**：
- ❌ 最初判断：哈梅内伊被杀 → 政权崩溃
- ✅ 修正判断：低估了威权政权韧性
- ✅ 更新预测：战争2-5年，窗口期延长

**错误原因分析**：

| 错误假设 | 实际情况 | 根本原因 |
|----------|----------|----------|
| 领导人死亡 = 政权崩溃 | ❌ 政权依然稳固 | 忽视了制度惯性 |
| 空袭+制裁 = 快速推翻 | ❌ 12年（伊拉克案例） | 忽视历史规律 |
| 经济压力 = 百姓造反 | ❌ 控制力强 | 低估控制能力 |

**核心教训**：

1. **不要低估威权政权的韧性**
   - 意识形态凝聚力（宗教、民族主义）
   - 暴力机器完整（军队、警察、情报）
   - 精英利益共同体（一荣俱荣，一损俱损）
   - 外部威胁反而强化内部团结

2. **没有地面战争，政权很难崩溃**
   - 历史证据：伊拉克12年、朝鲜70年、叙利亚13年、古巴60年
   - 空袭+制裁很少能推翻威权政权
   - 外部攻击反而强化合法性

3. **外部威胁强化内部团结**
   - 反美/反以 = 凝聚民心
   - 被侵略 = 民族主义高涨
   - 殉道文化 = 提升士气

4. **用户纠正是宝贵的学习机会**
   - 用户提供的洞察：组织结构紧凑、没有老百姓造反
   - 立刻修正判断，不要固执
   - 将错误转化为知识

**判断修正流程**：

```
错误判断 → 用户纠正 → 分析错误原因 → 查阅历史证据 → 更新判断 → 记录教训
```

**应用场景**：
- 以后分析任何威权政权时，首先考虑韧性因素
- 不要假设外部压力 = 快速崩溃
- 必须考虑：意识形态、暴力机器、精英团结、历史案例

**置信度调整**：
- 伊朗政权崩溃：从70% → 5%
- 战争持续时间：从"可能短期" → "2-5年（50%）"
- 台海窗口期：从2027年 → 2026下半年-2030年
