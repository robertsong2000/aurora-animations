## 📝 飞书文档工作流重要教训（2026-03-23）

### ⚠️ 核心问题

**错误理解**：`feishu_doc action=create content="..."` 会创建并写入内容

**正确理解**：`create` 只创建空白文档，`content` 参数无效

### ✅ 正确流程（三步走）

```bash
# 1. 创建文档
feishu_doc action=create title="标题"

# 2. 写入内容（必须！）
feishu_doc action=write doc_token=XXX content="完整内容"

# 3. 验证内容（必须！）
feishu_doc action=read doc_token=XXX
```

### 📋 验证标准

- ✅ `block_count > 1`（至少有内容块）
- ✅ `read` 返回的内容不为空
- ✅ `content` 字段包含预期内容

### 🔍 常见错误

| 错误 | 原因 | 解决方案 |
|------|------|----------|
| 文档为空 | 只调用 `create` | 必须调用 `write` |
| 内容没写入 | `content` 在 `create` 中无效 | 使用 `write` 或 `append` |
| 不知道是否成功 | 没有验证 | 必须用 `read` 验证 |
