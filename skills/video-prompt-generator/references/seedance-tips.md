# Seedance 使用技巧

## 关于 Seedance

**Seedance** 是火山引擎推出的AI视频生成工具，支持文生视频和图生视频。

### 特点
- 中文理解能力强
- 适合中国元素场景
- 人物动作自然
- 支持有声视频

---

## ⚠️ 重要限制

### 1. 字数限制
- **单个提示词：600字以内**
- **分镜脚本：800字以内**
- 精简但保留关键信息

### 2. 版权避让
- ❌ 禁止引用具体电影名称
- ❌ 禁止引用导演/演员名字
- ❌ 禁止引用受版权保护的作品
- ✅ 使用通用电影技术术语

### 3. 正确示例
✅ "35mm电影胶片质感，戏剧性光线"
✅ "电影级画质，高对比度"
✅ "专业电影机质感"

❌ "王家卫电影风格"
❌ "某某导演同款"
❌ "模仿某电影场景"

## 提示词优化技巧

### 1. 使用中文描述

Seedance 对中文理解很好，直接用中文描述效果更佳：

✅ 推荐：
```
一位穿着汉服的年轻女子在古色古香的庭院中漫步，春风拂面，
樱花飘落，阳光透过树叶洒下斑驳光影
```

❌ 不推荐：
```
A young woman in Hanfu walking in a traditional Chinese garden...
```

### 2. 结构化描述

按照固定结构组织提示词，效果更稳定：

```
[谁/什么] + [做什么] + [在哪里] + [怎么拍] + [什么光] + [什么风格]
```

### 3. 动作描述要具体

❌ 太简单：在走
✅ 具体：优雅地在铺满银杏叶的石板路上缓步前行

❌ 太简单：在看
✅ 具体：凝视远方，眼神中带着一丝期待

### 4. 场景要有细节

❌ 太简单：在公园
✅ 有细节：秋日的城市公园，银杏叶金黄，远处是现代建筑剪影，阳光透过树叶

### 5. 镜头运动要明确

| 效果 | 描述 |
|------|------|
| 聚焦 | 镜头缓缓推进，聚焦于... |
| 展现 | 镜头缓缓拉远，展现... |
| 跟随 | 镜头跟随主体移动 |
| 环绕 | 镜头环绕主体旋转 |
| 俯拍 | 无人机俯拍视角 |

## 常见问题

### Q: 画面模糊怎么办？

添加清晰度关键词：
- 4K高清
- 锐利清晰
- 高细节
- 专业画质

### Q: 动作不自然怎么办？

- 简化动作描述
- 添加"自然"、"流畅"等词
- 分解为简单动作

### Q: 风格不对怎么办？

明确指定风格关键词：
- 电影质感
- 纪录片风格
- Vlog日常
- 商业广告
- 艺术感

### Q: 光线太暗/太亮？

指定光线类型：
- 黄金时刻（暖调）
- 蓝调时刻（冷调）
- 柔和自然光
- 专业影棚光
- 霓虹灯光

## 注意事项

1. **提示词长度** - 建议控制在50-100字，太长可能效果不稳定
2. **避免冲突** - 不要同时指定相反的风格（如"冷调"+"暖调"）
3. **具体优先** - 越具体的描述效果越好
4. **中文优先** - Seedance对中文理解更好
5. **测试迭代** - 多尝试不同表达方式，找到最佳效果
