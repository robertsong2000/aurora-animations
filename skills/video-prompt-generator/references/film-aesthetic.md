# 胶片质感参考（Seedance 中文版）

## 常用胶片类型

### 柯达 Kodak

#### Portra 400（最常用）
**特点**：
- 细腻柔和颗粒
- 暖调金黄色调
- 肤色还原自然
- 饱和度适中
- 宽容度高

**适用场景**：
- 人像摄影
- 婚礼/度假
- 日出日落
- 暖色调场景

**描述示例**：
```
35mm柯达Portra 400胶片质感，细腻柔和颗粒，暖调金黄阳光色调，
肤色自然，梦幻度假氛围
```

#### Ektar 100
**特点**：
- 极细颗粒
- 高饱和度
- 鲜艳色彩
- 适合风景

**描述示例**：
```
Ektar 100胶片质感，极细颗粒，高饱和度鲜艳色彩，风景摄影级画质
```

#### Gold 200
**特点**：
- 颗粒适中
- 暖色调
- 日常感
- 性价比高

**描述示例**：
```
Kodak Gold 200，温暖日常色调，自然颗粒感，Vlog风格
```

### 富士 Fuji

#### C200
**特点**：
- 冷调偏青
- 颗粒细腻
- 日系清新感

**描述示例**：
```
Fuji C200胶片质感，冷调清新，日系胶片感，细腻颗粒
```

#### Superia 400
**特点**：
- 色彩鲜艳
- 颗粒明显
- 复古感

**描述示例**：
```
Fuji Superia 400，鲜艳色彩，复古胶片颗粒，街头摄影风格
```

---

### 数字电影质感

#### Arri Alexa
**特点**：
- 电影级画质
- 高动态范围
- 自然色彩
- 柔和高光

**描述示例**：
```
Arri Alexa电影级质感，高动态范围，自然色彩还原，柔和逆光处理
```

#### RED 8K
**特点**：
- 超高分辨率
- 锐利细节
- 高对比度
- 科幻感

**描述示例**：
```
RED 8K数字电影机质感，超高分辨率，锐利细节，高对比度，科技电影感
```

## 对比度描述

| 对比度 | 描述 | 效果 |
|--------|------|------|
| 低对比度 | 柔和、梦幻、胶片感 | 复古梦幻 |
| 中等对比度 | 自然、平衡 | 日常场景 |
| 高对比度 | 锐利、戏剧性、电影感 | 电影大片 |
| 极高对比度 | 强烈、冲击、艺术 | 艺术影片 |
