# 视频风格参考

## 1. 电影感 (Cinematic)

**特点**：高质感、叙事性强、情绪饱满

**镜头语言**：
- 浅景深、焦外虚化
- 缓慢推拉摇移
- 电影级调色（低对比度、柔和色调）
- 镜头光晕、眩光效果

**适用场景**：品牌故事、情感短片、产品高端展示

**提示词关键词**：
```
cinematic, film look, shallow depth of field, 
anamorphic lens, color graded, movie quality, 
dramatic lighting, slow motion
```

---

## 2. Vlog日常 (Casual Vlog)

**特点**：真实、亲切、生活化

**镜头语言**：
- 手持感、略带晃动
- 自拍视角
- 快速剪辑、跳剪
- 自然光线

**适用场景**：日常分享、旅行记录、生活vlog

**提示词关键词**：
```
vlog style, handheld camera, natural lighting,
casual, authentic, selfie angle, jump cuts,
day in the life
```

---

## 3. 搞笑/喜剧 (Comedy)

**特点**：夸张、节奏快、反转

**镜头语言**：
- 夸张表情特写
- 快速切换
- 意外转场
- 音效配合

**适用场景**：段子、搞笑剧情、生活吐槽

**提示词关键词**：
```
comedic timing, funny, exaggerated expressions,
slapstick, humorous, playful, quick cuts,
unexpected twist
```

---

## 4. 知识科普 (Educational)

**特点**：清晰、专业、易懂

**镜头语言**：
- 屏幕录制 + 人像画中画
- 图表动画
- 重点标注
- 章节分明

**适用场景**：教程、科普、干货分享

**提示词关键词**：
```
educational, tutorial style, clear visuals,
infographics, step by step, professional,
informative, easy to understand
```

---

## 5. 产品展示 (Product Showcase)

**特点**：精致、突出产品、引导购买

**镜头语言**：
- 360度展示
- 细节特写
- 使用场景
- 前后对比

**适用场景**：电商、产品推广、开箱测评

**提示词关键词**：
```
product photography, clean background,
studio lighting, detailed close-ups,
360 rotation, lifestyle shots,
commercial quality
```

---

## 6. 情感治愈 (Emotional/Healing)

**特点**：温暖、慢节奏、引发共鸣

**镜头语言**：
- 柔和光线
- 慢镜头
- 自然元素（阳光、风、水）
- 温暖色调

**适用场景**：情感故事、治愈内容、心灵鸡汤

**提示词关键词**：
```
emotional, heartwarming, soft lighting,
slow motion, warm tones, peaceful,
sentimental, touching moments
```

---

## 7. 快节奏剪辑 (Fast-paced/Montage)

**特点**：节奏强、视觉冲击、卡点

**镜头语言**：
- 极快切换（0.5-1秒/镜头）
- 音乐卡点
- 转场特效
- 运动镜头

**适用场景**：舞蹈、运动、时尚、混剪

**提示词关键词**：
```
fast-paced, dynamic editing, beat sync,
quick cuts, energetic, high tempo,
montage, visual rhythm
```

---

## 8. 动画/卡通 (Animation)

**特点**：创意、夸张、不受限

**风格类型**：
- 2D扁平动画
- 3D动画
- 手绘风格
- 白板动画

**适用场景**：解说、故事、品牌动画

**提示词关键词**：
```
animated, cartoon style, 2D/3D animation,
playful characters, vibrant colors,
hand-drawn, motion graphics
```

---

## 9. 复古/怀旧 (Retro/Vintage)

**特点**：年代感、胶片质感、温暖

**视觉元素**：
- 胶片颗粒
- 暖黄/棕色调
- 划痕效果
- 老照片滤镜

**适用场景**：怀旧主题、复古风格、回忆录

**提示词关键词**：
```
vintage, retro aesthetic, film grain,
warm sepia tones, nostalgic, analog look,
VHS effect, old school
```

---

## 10. 科技/未来感 (Tech/Futuristic)

**特点**：酷炫、高科技、赛博朋克

**视觉元素**：
- 霓虹灯光
- 全息效果
- 蓝紫色调
- 粒子特效
- 故障艺术

**适用场景**：科技产品、游戏、未来主题

**提示词关键词**：
```
futuristic, cyberpunk, neon lights,
holographic effects, sci-fi, tech aesthetic,
glitch art, digital, high-tech
```

---

## 风格组合示例

| 组合 | 效果 |
|------|------|
| 电影感 + 情感治愈 | 高质感情感短片 |
