# 平台特点参考

## 国内平台

### 抖音 (Douyin)

**视频时长**：15秒-5分钟（推荐15-60秒）

**内容特点**：
- 强调前3秒"黄金开头"
- 喜欢反转、悬念、冲突
- 音乐卡点、节奏感强
- 贴纸、特效丰富

**推荐风格**：
- 搞笑/剧情
- 快节奏剪辑
- 知识科普（干货）
- 情感治愈

**文案特点**：
- 口语化、接地气
- 引导互动："双击关注不迷路"
- 热梗、流行语

**标题公式**：
```
数字+悬念：3个方法让你月入过万
冲突+反转：没想到结局是这样
提问+好奇：你知道为什么...吗
```

---

### 快手 (Kuaishou)

**视频时长**：15秒-10分钟

**内容特点**：
- 更接地气、真实
- 老铁文化、强调信任
- 直播导流强

**推荐风格**：
- 生活记录
- 才艺展示
- 搞笑段子
- 产品带货

---

### B站 (Bilibili)

**视频时长**：1分钟-30分钟（中长视频为主）

**内容特点**：
- 弹幕文化
- 深度内容
- 二次元、游戏、科技
- 知识科普、测评

**推荐风格**：
- 知识科普
- 游戏解说
- 动漫相关
- 测评/开箱

**文案特点**：
- 专业、有深度
- 引用梗文化
- "三连"引导

---

### 小红书 (Xiaohongshu)

**视频时长**：15秒-5分钟

**内容特点**：
- 种草、分享
- 精致生活
- 女性用户为主
- 图文+视频结合

**推荐风格**：
- 美妆/穿搭
- 探店/美食
- 旅行攻略
- 生活方式

**文案特点**：
- 情绪化表达
- 详细攻略
- "姐妹们"亲切称呼

---

## 海外平台

### TikTok

**视频时长**：15秒-10分钟（推荐15-60秒）

**内容特点**：
- 全球化
- 音乐驱动
- 挑战赛（Challenge）
- Trending话题

**推荐风格**：
- 舞蹈/音乐
- 搞笑短剧
- 生活小技巧
- 创意转场

**推荐风格**：

- 搞笑
- 幕后花絮

---

## 跨平台内容策略

### 一鱼多吃

一个素材可适配多个平台：

```
原始素材（3分钟）
    ├── 抖音：剪辑成3个15秒短视频
    ├── B站：保留完整版
    ├── 小红书：提取精华1分钟版
    └── 视频号：中等长度2分钟版
```

### 平台适配要点

1. **时长调整** - 根据平台裁剪
2. **封面设计** - 不同平台尺寸要求
3. **文案修改** - 符合平台调性
4. **发布时间** - 各平台高峰期不同
