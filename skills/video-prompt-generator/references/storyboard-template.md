## 分镜脚本示例

### 示例1：奢华度假早餐（12秒）

```markdown
## 视频标题：萝卜头的大溪地奢华早餐
时长： 12秒 | 风格： 热带奢华度假 | 平台： 抖音/TikTok

### 🎨 整体视觉风格（关键！）
**胶片质感：** 35mm柯达Portra 400，细腻柔和颗粒，暖调金黄阳光色调，饱和度略高，梦幻度假氛围
**色调基调：** 金黄阳光 + 宝石蓝泻湖 + 翠绿椰树 + 白色建筑，高饱和度暖色调
**音乐：** 大溪地传统海螺号角开场，转轻柔乌克丽丽配海浪声

---

### 分镜1（0-3秒）【开场 - 苏醒】

**画面：**
- 中景，萝卜头躺水上别墅大床，金色晨光穿透白纱窗帘洒在脸上
- 玻璃地板特写：热带鱼游过
- 远景推近：莫努岛山峰晨雾缭绕
- 突然——远处传来悠长海螺号角"呜——"

**胶片质感：** Portra 400，暖金晨光，柔和逆光晕

**配音：** "清晨6点，大溪地波拉波拉..."

**音乐/音效：** 海螺号角声 + 海浪轻拍

---

### 分镜2（3-8秒）【核心 - 独木舟送餐】

**画面：**
- 远景：晨雾中划来传统独木舟(va'a)，船身雕刻图腾
- 中景推进：4位土著服务人员——男士穿橙红花pareu戴椰叶头饰，女士戴鸡蛋花发饰黑珍珠项链，脚踝贝壳链叮当作响
- 特写：领头者高举大海螺再次吹响
- 侧拍：女士捧芭蕉叶包裹的托盘，优雅步上露台
- 快剪：展开芭蕉叶，露出椰子、可颂、热带水果拼盘、班尼迪克蛋
- 微距特写：流心蛋黄切开瞬间，金色蛋液缓缓流出

**胶片质感：** Portra 400，高饱和度，暖调 + 宝石蓝泻湖对比色

**配音：** "海螺号角响起，独木舟送来天堂早餐！"

**音乐/音效：** 乌克丽丽轻快旋律 + 贝壳叮当声

---

### 分镜3（8-12秒）【结尾 - 品尝】

**画面：**
- 中景：萝卜头坐藤编椅，手持劈开的新鲜椰子喝椰汁，满足表情
- 特写：咬可颂，酥脆外皮掉渣，黄油香气可视化（热气缭绕）
- 拉远景：萝卜头背后是宝石蓝泻湖、远处山峰、阳光洒满整个画面
- 字幕淡入："这才是生活 🥕✨"

**胶片质感：** Portra 400，梦幻暖调，轻微镜头光晕

**配音：** "如果每天都能这样醒来..."

**音乐/音效：** 乌克丽丽渐弱 + 海浪声

---

### 📸 拍摄要点：
- 航拍开场：水上别墅俯瞰 → 推近露台
- 慢动作：独木舟划过平静水面，涟漪扩散
- 微距特写：食物细节（流心蛋、椰肉、水果纹理）
- 自然光：充分利用清晨金色阳光，逆光拍摄增强梦幻感
- 音效设计：海螺号角 + 海浪 + 贝壳叮当 + 乌克丽丽 + 咬可颂酥脆声
- 色调统一：所有镜头保持Portra 400暖调，避免冷调破坏度假氛围
```

```markdown
高对比度，冷调

### 分镜3（10-15秒）【结尾 - 点亮】

- 特写：屏幕亮起，显示Hello

- 字幕淡入："年度最佳 📱"

**胶片质感：** 屏幕冷光与环境对比，电影感

**配音：** "这就是...未来"

**音乐/音效：** 科技音效渐弱

### 📸 拍摄要点：
- 纯黑背景突出产品
- 慢动作展现仪式感
- 微距捕捉材质细节
- 冷色调保持科技感
- 音效增强开箱仪式感
```

## 常用分镜结构

### 三段式（最常用）
```
分镜1（0-3秒）：开场 - 吸引注意力
分镜2（3-9秒）：核心 - 展示主体内容
分镜3（9-12秒）：结尾 - 总结/互动引导
```

### 四段式
```
分镜1（0-2秒）：黄金开头 - 悬念/冲突
分镜2（2-6秒）：发展 - 核心内容
分镜3（6-10秒）：高潮 - 最精彩部分
分镜4（10-12秒）：结尾 - 互动引导
```

### 五段式（较长视频）
```
分镜1：开场钩子
分镜2：铺垫背景
分镜3：核心展示
分镜4：情感升华
分镜5：结尾引导
```
