# 镜头语言详解

## 景别

### 基本景别

| 景别 | 英文 | 描述 | 用途 |
|------|------|------|------|
| **远景** | Extreme Wide Shot (EWS) | 环境为主，人物很小 | 展现场景、建立空间感 |
| **全景** | Wide Shot (WS) | 人物全身+周围环境 | 展示人物与空间关系 |
| **中景** | Medium Shot (MS) | 人物腰部以上 | 对话、日常场景 |
| **中近景** | Medium Close-Up (MCU) | 胸部以上 | 情感表达、对话 |
| **近景** | Close-Up (CU) | 肩膀以上，面部为主 | 情绪、表情 |
| **特写** | Extreme Close-Up (ECU) | 局部放大（眼睛/嘴唇/物品） | 强调细节、情感冲击 |
| **微距** | Macro | 极近距离拍摄 | 纹理、材质、食物细节 |

### 景别组合示例

**开场所需**：
```
远景（建立场景）→ 全景（展示环境）→ 中景（引入人物）
```

**情感强调**：
```
中景 → 近景 → 特写（情绪递进）
```

**产品展示**：
```
全景（产品整体）→ 中景（使用场景）→ 特写（细节）→ 微距（材质纹理）
```

## 运镜方式

### 基本运镜

| 运镜 | 英文 | 描述 | 效果 |
|------|------|------|------|
| **推** | Dolly In / Push In | 镜头向前移动 | 聚焦、强调、压迫感 |
| **拉** | Dolly Out / Pull Back | 镜头向后移动 | 展现环境、释放感 |
| **左摇** | Pan Left | 镜头水平左转 | 展示空间、跟随运动 |
| **右摇** | Pan Right | 镜头水平右转 | 展示空间、跟随运动 |
| **上摇** | Tilt Up | 镜头垂直上转 | 展现高度、仰望感 |
| **下摇** | Tilt Down | 镜头垂直下转 | 展现深度、俯视感 |
| **跟随** | Tracking / Follow | 镜头跟随主体移动 | 沉浸感、参与感 |
| **环绕** | Orbit / Arc | 围绕主体旋转 | 360度展示、仪式感 |
| **升降** | Crane / Boom | 镜头上下移动 | 电影感、大场面 |

### 高级运镜

| 运镜 | 描述 | 用途 |
|------|------|------|
| **推拉变焦** | Dolly Zoom | 推镜头同时变焦拉长 | 眩晕感、情绪冲击 |
| **手持晃动** | Handheld Shake | 模拟手持不稳定感 | 真实感、紧张感 |
| **稳定器跟拍** | Gimbal Follow | 平滑跟随 | Vlog、运动场景 |
| **航拍俯瞰** | Aerial / Drone | 无人机俯拍 | 大场景、震撼开场 |
| **穿越机视角** | FPV | 高速穿越 | 极限运动、刺激感 |

## 镜头角度

| 角度 | 英文 | 描述 | 心理效果 |
|------|------|------|----------|
| **平视** | Eye Level | 与主体视线平齐 | 平等、客观、日常 |
| **仰视** | Low Angle | 从低处向上拍 | 威严、强大、压迫 |
| **俯视** | High Angle | 从高处向下拍 | 渺小、脆弱、全局 |
| **鸟瞰** | Bird's Eye | 垂直向下俯拍 | 全局、上帝视角 |
| **斜角** | Dutch Angle | 倾斜构图 | 不安、紧张、诡异 |
| **主观视角** | POV | 第一人称视角 | 代入感、沉浸感 |

## 焦距与景深

### 焦距分类

| 焦距 | 类型 | 特点 | 用途 |
|------|------|------|------|
| 14-24mm | 超广角 | 夸张透视、大景深 | 风景、建筑、vlog |
| 24-35mm | 广角 | 自然透视、环境感 | 街拍、日常、vlog |
| 50mm | 标准 | 接近人眼视角 | 人像、对话场景 |
| 85-135mm | 中长焦 | 压缩空间、浅景深 | 人像特写、情感 |
| 200mm+ | 长焦 | 强烈压缩、极浅景深 | 远距离拍摄、电影感 |

### 景深控制

**浅景深（虚化背景）**：
```
- 大光圈（f/1.4 - f/2.8）
- 长焦距
- 主体靠近镜头
- 背景远离主体

描述：shallow depth of field, bokeh, background blur
```

**深景深（清晰背景）**：
```
- 小光圈（f/8 - f/16）
- 广角镜头
- 主体远离镜头

描述：deep focus, everything in focus
```

## 运镜描述示例

### 推镜头

```
中景，镜头缓缓推近人物面部，捕捉表情变化
```

```
远景，快速推进至特写，强调关键时刻
```

### 拉镜头

```
特写，镜头缓缓拉远，展现人物所处环境
```

```
中景，突然拉开至远景，揭示惊喜场景
```

```
手持跟拍，镜头跟随人物穿过拥挤街道
```

```
稳定器跟拍，平滑跟随奔跑的人物
```

### 环绕镜头

```
镜头围绕主体360度旋转，展现完整场景
```

```
环绕拍摄，阳光在主体周围流转
```

## 镜头组合技巧

### 情绪递进

```
全景 → 中景 → 近景 → 特写
（情绪逐渐加强）
```

### 场景切换

```
远景（场景A）→ 切 → 远景（场景B）
（清晰的空间转换）
```

### 时间流逝

```
同一角度不同光线：晨光 → 正午 → 黄昏
（时间压缩）
```

### 对比剪辑

```
特写（细节A）→ 特写（细节B）→ 特写（细节C）
（快节奏展示多个细节）
```
